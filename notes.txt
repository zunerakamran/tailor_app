-> Uploading the images on google drive
a. Enable Google Drive API
1. Go to https://console.cloud.google.com/?pli=1
2. Create new project
3. Navigate to APIs & Services → Library → Google Drive API → Enable
4. Go to APIs & Services → Credentials → Create credentials → Service account.
5. Download the JSON key file (service_account.json).

b. Install Require Python Library
   py -m pip install --upgrade google-api-python-client google-auth-httplib2 google-auth-oauthlib
